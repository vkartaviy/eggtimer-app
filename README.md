# Simple Egg Timer App for Pomodoro Technique

## Installation

Download and install the relevant distribution from the [releases page](http://github.com/vkartaviy/eggtimer-app/releases).

## Development

```
yarn install
yarn start
```

#### Building a distribution

```
yarn build
```
